# Telecaller Agent Script App

A dedicated Flutter application designed for telecallers handling campaigns for events such as **Elastic Agentic AI Day – Bangalore**.

## Features
- **Interactive Script Viewer:** Clean, step-by-step reading flow covering Greetings, Event Details, Highlights, and Closing remarks.
- **Integrated Lead Capture:** A side panel (or bottom section on mobile) to instantly capture client data (Name, Email, Company, Job Title) if they show interest.
- **Responsive Layout:** Works beautifully on desktop/tablet for call center agents, and stacks perfectly on mobile screens for agents on the go.

## Built With
- Flutter
- Material 3 Design

## Setup & Run
To run this project:
1. Ensure you have Flutter installed.
2. Run `flutter pub get`.
3. Start the app via `flutter run`.
